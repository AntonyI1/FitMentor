# FitMentor - AI-Powered Fitness Coach

An intelligent fitness platform that combines AI-driven calorie calculation and personalized workout recommendations.

## Features

### 1. Calorie Maintenance Calculator
- Accurate calorie calculation based on personal metrics
- Macronutrient breakdown (protein, carbs, fats)
- Powered by TensorFlow ML models
- Personalized based on fitness goals

### 2. AI Workout Suggester
- Personalized workout plan generation
- Considers experience level, equipment, and schedule
- Exercise recommendations with images
- Sets, reps, and progression tracking
- Powered by TensorFlow and PyTorch models

## Tech Stack

- **Backend**: Python Flask API
- **ML Models**: TensorFlow, PyTorch
- **Frontend**: HTML, CSS, JavaScript
- **Data**: NumPy, Pandas

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Run backend server: `python backend/app.py`
3. Open `frontend/index.html` in your browser
